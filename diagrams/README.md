# Architecture & Sequence Diagrams

This directory contains **Mermaid diagrams** representing system architectures
and request lifecycles used across the repository.

## Diagram Types
- System architecture diagrams
- RAG pipelines
- MLOps workflows
- Sequence diagrams

## Purpose
- Communicate system design clearly
- Document architectural decisions
- Support interviews and design discussions

## Note
Diagrams reflect **production patterns**, not toy examples.
